# TDD Kit - JavaScript BDD

## Initialization

    npm install

## Run unit tests

    grunt test

## Run lint, unit tests and coverage

    grunt

## Libraries documentation

* Assertion api: [chai BDD api]
* Test runner: [mocha cli]
* Linter: [jshint]
* Coverage: [istanbul]
* Mock api: [sinonjs]
* BDD Mock api: [sinon-chai]
